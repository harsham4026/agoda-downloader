1) This is AGODA project written in Python
2) Download the program and and Run following command
         a) python application.py
3) Run unittest using following command
         a) pytest
4) To create python environment do following
         a) python -m virtualenv agoda
         b) source agoda/bin/activate
         c) pip install -r requirements.txt

5) Project is Dockerised
         a) docker build -t agoda .

6) Once program started it will ask for Output directory. Make sure path has all permission

7) Enter url,username,password in the file site_details line by line.(Example file included)
